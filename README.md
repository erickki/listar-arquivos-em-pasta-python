# Listar arquivos em pasta Python

## Descrição

Este repositório contém um programa simples e direto em Python, projetado para automatizar a tarefa de listar arquivos de um diretório. O usuário informa o caminho de uma pasta, e o script processa todos os arquivos existentes nela, gerando um arquivo de texto (.txt) contendo a relação completa de nomes encontrados.

A ferramenta é útil para quem precisa organizar, catalogar, auditar ou documentar o conteúdo de pastas extensas, evitando a necessidade de listagens manuais. Pode ser utilizada em ambientes domésticos, profissionais, técnicos ou educacionais, com foco em praticidade e rapidez.

## Funcionalidades

- Leitura do caminho de uma pasta inserido pelo usuário.
- Geração automática de um arquivo .txt com a lista dos arquivos encontrados.
- Suporte a qualquer tipo de extensão de arquivo.
- Tratamento básico de erros (como caminho inválido ou pasta inexistente).
- Execução rápida e simples, sem dependências externas.

## Vantagens

- Facilita auditorias de arquivos.
- Evita listagem manual de diretórios.
- Totalmente automatizado e fácil de usar.
- Pode ser adaptado para novos formatos ou integrações.
- Código simples, ideal para estudo ou aplicação prática.
